# Images Folder

## Required Images

### 1. Founder Image
- **Filename:** `founder.jpg`
- **Location:** This folder (`images/founder.jpg`)
- **Recommended size:** 350x450px or similar portrait orientation
- **Format:** JPG or PNG
- **Displayed in:** "About the Founder" section

### 2. Investment Partner Image
- **Filename:** `investment-partner.jpg`
- **Location:** This folder (`images/investment-partner.jpg`)
- **Recommended size:** 350x450px or similar portrait orientation
- **Format:** JPG or PNG
- **Displayed in:** "Investment Partner" section

## How to Add Images

1. Save the photos with the exact filenames:
   - `founder.jpg` (Mohammad Gelani Pasha)
   - `investment-partner.jpg` (Mohd Shahnawaz Pasha)
2. Place both files in this `images` folder
3. The website will automatically display them

**Note:** If images are not found, a placeholder will be shown instead.
